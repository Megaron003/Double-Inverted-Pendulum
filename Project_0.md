# 🚀 Pitch do Projeto: Análise de Dataset de Sistema Não-Linear Caótico
**Grupo: 3** Guilherme Braga do Vale & Lucas Sanches Garcia

---

### 1. A Pergunta
> O conjunto de dados descreve um sistema deterministico e permite, após pré-processamento, a implementação de uma rede neural preditiva capaz de identificar o modelo e controlar o sistema?

### 2. Evidência Visual
> A visualização dos dados se deu através da geometria de pontos o que permitiu averiguar e denotar a natureza não linear caótica do sistema em questão o que direcionou a análise para processos mais profundos e especificamente para sistemas não lineares.

### 3. Limitação Marcante
> O uso de métodos adequados que permitissem averiguar se o sistema era determinístico e não puramente estocástico.

### 4. Evolução Técnica
> O projeto zero proporcionou uma boa compreensão na aquisição e tabulação em formato Tidy dos dados. Esse processo favoreceu a compreensão de quais dados devem ser preservados, priorizados ou mesmo alterados, a exemplo do cálculo de seno e cosseno dos ângulos theta_1 e theta_2.

### 5. Próximo Passo
> Realizar o levantamento de hipóteses para compreeder com mais clareza a natureza dos dados e assim estiular possibilidades futuras para de suas aplicações.
